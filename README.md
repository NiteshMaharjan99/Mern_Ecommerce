# Ecommerce Website

Made with MERN stack

## ⭐️ Features

- Authentication
- Product reviews
- Pagination
- Payment(PayPal) integration
- User Profile(Update/Edit)
- User orders(All the items ordered)
- Product management(Add/Edit - Admin only)
- Orders management(mark as delivered - Admin only)

## 💻 Tech Stack

- MERN stack
- Redux
- React Bootstrap
